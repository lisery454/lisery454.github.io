本文记录了一些常用的C#代码XML注释的用法

常用

summary

描述方法做了什么

remark

用于补充一些信息

记录成员

returns

记录返回值

param

记录输入参数

paramref

用来显示参数名字

exception

用于记录可能抛出的异常

value

感觉类似于remark，只不过换了个名字

设置文档输出格式

para

用于创建一个双空格段落

br

创建一个单空格空行

list

用于表示一个列表，type可以是bullet，number，table

c / code

用于表示代码和代码块

example

用于介绍例子，通常和code一起使用
